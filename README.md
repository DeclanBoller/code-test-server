## Install

This project uses [node](http://nodejs.org) and [npm](https://npmjs.com). You will need to have them locally installed to run this project.

Clone this Repo: 

```
$ git clone https://github.com/DeclanBoller/code-test-server.git
```

Run NPM Install: 

```
$ npm install 
```

Run Server

```
$ node server.js
```
